# Planning Phase: CMMC Readiness Baseline

## 1. Project Scope Statement
**Goal:** Achieve CMMC Level 2 "Ready" status for the mock Enclave.
* **In-Scope:** NIST 800-171 controls, SSP documentation, and MFA implementation.
* **Out-of-Scope:** Marketing department hardware and guest networks.
* **Constraints (EEF):** Must be completed before the contract renewal date in 6 months.

---

## 2. Work Breakdown Structure (WBS)
*Decomposing the project into manageable work packages.*

- **1.0 Gap Analysis**
  - 1.1 Review Current Policies
  - 1.2 Identify Technical Gaps
- **2.0 Technical Implementation**
  - 2.1 Access Control (MFA)
  - 2.2 Identification & Authentication
  - 2.3 Media Protection (Encryption)
- **3.0 Documentation**
  - 3.1 Draft System Security Plan (SSP)
  - 3.2 Create POA&M (Plan of Action & Milestones)
- **4.0 Internal Assessment**
  - 4.1 Dry-run Audit
