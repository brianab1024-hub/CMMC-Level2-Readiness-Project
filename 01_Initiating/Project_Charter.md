# Project Charter: CMMC Level 2 Readiness
**Project Name:** Aspire CMMC Certification  
**Project Manager:** Briana Bellamy, MBA  
**Status:** Draft / Initiating  

---

## 1. Project Purpose & Business Case
To achieve CMMC Level 2 certification to remain eligible for Department of Defense (DoD) contracts. This project is a response to an **External EEF** (Regulatory Requirements).

## 2. High-Level Requirements
* Implement all 110 NIST 800-171 security controls.
* Document a System Security Plan (SSP).
* Establish a Plan of Action and Milestones (POA&M) for any gaps.

## 3. Project Objectives & Success Criteria
| Objective | Success Criteria |
| :--- | :--- |
| **Security Compliance** | 100% of controls verified by internal audit. |
| **Certification** | Successful C3PAO assessment by Q4 2026. |
| **Budget** | Project completed within the $150,000 baseline. |

## 4. High-Level Risks
* **Internal EEF:** Limited availability of senior IT staff for configuration updates.
* **External EEF:** Long lead times for specialized encrypted hardware.

## 5. Summary Milestone Schedule
1. **Initiation:** Project Charter Signed (Today)
2. **Planning:** Gap Analysis & Budget Approval (Month 1)
3. **Execution:** Control Implementation (Months 2-5)
4. **Closing:** Final Audit & Lessons Learned (Month 6)

## 6. Organizational Process Assets (OPAs) to be Used
* Company Project Management Policy.
* Incident Response Plan Template.
* Lessons learned from previous ISO audits.
